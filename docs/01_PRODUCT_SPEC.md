# Valorant LFG Mini App — Product Spec

## 1. Идея проекта

Telegram Mini App для Valorant-комьюнити. Пользователь заходит в Telegram-бота, оттуда открывает мини-приложение и может:

1. Заполнить анкету.
2. Найти команду.
3. Найти игрока.
4. Открыть помощь.
5. Изменить настройки.

Цель проекта — заменить хаотичные сообщения в чате типа “ищу дуо, пишите” на структурированный поиск по параметрам: ранг, роль, город, микрофон, режим, цель и вайб.

## 2. Общий пользовательский путь

```text
Пользователь открывает Telegram-бота
↓
Нажимает “Открыть приложение”
↓
Попадает в Telegram Mini App
↓
Видит главный экран с 5 разделами
↓
Создает анкету
↓
Ищет команду или отдельного игрока
↓
Отправляет заявку/приглашение
↓
Вторая сторона принимает или отклоняет
↓
Пользователи получают контакты / Riot ID
```

## 3. Главные разделы Mini App

### 3.1 Анкета

Раздел профиля игрока. На базе анкеты строится весь поиск.

Финальные поля анкеты:

```text
Ник в Valorant
Riot ID
Ранг
Роль
Город
Возраст
Микрофон
Любимый режим
Цель
О себе
```

Важно: вместо “Регион” используется **Город**.

Обязательные поля:

```text
Ник в Valorant
Riot ID
Ранг
Роль
Город
Микрофон
Любимый режим
Цель
```

Необязательные поля:

```text
Возраст
О себе
```

Функции анкеты:

- Создать анкету.
- Редактировать анкету.
- Скрыть/показать анкету в поиске.
- Предпросмотр анкеты.
- Валидация Riot ID в формате `Nick#Tag`.
- Прогресс заполнения профиля.
- Статус анкеты: active, hidden, moderation, blocked.

### 3.2 Найти команду

Раздел для поиска готовой команды/стака или создания своей команды.

Пользователь может:

- Смотреть список команд.
- Фильтровать команды.
- Открывать карточку команды.
- Создавать команду.
- Откликаться в команду.
- Управлять своей командой, если он капитан.
- Принимать/отклонять заявки в свою команду.

Фильтры поиска команды:

```text
Ранг / диапазон ранга
Нужная роль
Город / формат онлайн
Режим
Микрофон
Цель
Время игры
```

Поля команды:

```text
Название команды
Описание
Ранг команды / диапазон
Нужные роли
Город / формат
Режимы
Количество игроков сейчас
Сколько игроков ищем
Требование к микрофону
Время игры
Цель команды
Статус
Видимость
Капитан
```

### 3.3 Найти игрока

Раздел для поиска отдельного игрока: дуо, трио, пятого в стак или просто тиммейта.

Пользователь может:

- Смотреть список игроков.
- Фильтровать игроков.
- Открывать подробную анкету.
- Отправлять приглашение сыграть.
- Смотреть входящие/исходящие приглашения.
- Принимать/отклонять приглашения.
- Жаловаться/блокировать игроков в будущем.

Фильтры поиска игрока:

```text
Ранг
Роль
Город
Микрофон
Любимый режим
Цель
Возраст
```

Быстрые пресеты, которые стоит добавить/улучшить:

```text
Найти дуо
Найти трио
Ranked сейчас
Premier
Из моего города
Только с микро
```

## 4. Тон продукта

Проект для Valorant-комьюнити, поэтому тон интерфейса может быть живым, коротким и немного ироничным, но без перегиба.

Примеры:

```text
Найди тиммейтов без хаоса в чате.
Анкета → поиск → приглашение → катка.
Скрыть анкету, если сегодня хочется соло-страдать.
```

Не стоит делать интерфейс слишком официальным — это игровое комьюнити.
