# Prompt для нового ИИ

Скопируй этот текст новому ИИ вместе со всеми `.md` файлами и кодом проекта.

---

Ты помогаешь продолжить разработку Telegram Mini App для Valorant LFG-комьюнити. Проект называется `valorant-lfg-miniapp`.

Контекст: у пользователя есть Telegram-канал/чат по Valorant, где люди ищут игроков в команду и общаются. Вместо обычного Telegram-бота принято решение делать Telegram Mini App: пользователь заходит в бота, нажимает кнопку открытия приложения и попадает в мини-приложение.

Уже согласованы основные разделы Mini App:

1. **Анкета**
2. **Найти команду**
3. **Найти игрока**
4. **Помощь**
5. **Настройки**

На текущий момент в коде реализован стартовый MVP-каркас:

- `apps/backend` — FastAPI API, SQLAlchemy models, auth через Telegram initData, seed.
- `apps/bot` — aiogram bot, который открывает Mini App.
- `apps/web` — React/Vite frontend для Mini App.

Помощь и Настройки пока заглушки/минимальная версия. Их нужно пройти позже.

Важное требование: продолжай от уже согласованной структуры, не меняй концепцию без причины. Если нужно добавить новый функционал, расширяй текущую архитектуру.

Сначала прочитай файлы:

1. `docs/01_PRODUCT_SPEC.md`
2. `docs/02_UX_FLOWS.md`
3. `docs/03_DATABASE_SCHEMA.md`
4. `docs/04_API_SPEC.md`
5. `docs/05_TECH_ARCHITECTURE.md`
6. `docs/06_IMPLEMENTATION_STATUS.md`
7. `docs/07_ROADMAP.md`
8. `docs/CODE_DUMP.md`

После этого помогай пользователю дописывать код, улучшать UX, добавлять разделы “Помощь” и “Настройки”, модерацию, уведомления, админку и деплой.
