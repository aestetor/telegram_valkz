# UX Flows

## 1. Главный экран

Главный экран Mini App содержит 5 карточек:

```text
👤 Анкета
Заполни профиль, чтобы тебя могли найти

👥 Найти команду
Подбери стак или команду под свой ранг

🎯 Найти игрока
Найди дуо, трио или пятого в пати

❓ Помощь
Как пользоваться приложением

⚙️ Настройки
Приватность, уведомления, город, язык
```

Если у пользователя нет анкеты, после загрузки Mini App его можно сразу направлять на раздел “Анкета”.

## 2. Анкета

### 2.1 Первый вход

```text
Пользователь открывает Mini App
↓
Backend пытается найти профиль
↓
Если профиля нет — открыть экран анкеты
↓
Пользователь заполняет обязательные поля
↓
Сохраняет профиль
↓
Видит превью и процент заполнения
```

### 2.2 Заполнение анкеты

Поля:

```text
Ник в Valorant — input
Riot ID — input с валидацией Nick#Tag
Ранг — select
Роль — multiselect
Город — input
Возраст — select, необязательно
Микрофон — select
Любимый режим — multiselect
Цель — multiselect
О себе — textarea, необязательно
Показывать в поиске — toggle
```

### 2.3 Превью анкеты

Пример отображения:

```text
👤 ShadowPeek

🎮 Riot ID: Shadow#777
🏆 Ранг: Platinum
🎭 Роль: Controller / Sentinel
🏙 Город: Алматы
🎂 Возраст: 18-21
🎙 Микрофон: Да
🕹 Любимый режим: Ranked
🎯 Цель: Найти дуо / Найти команду

💬 О себе:
Играю вечером, не токсик. Мейн Omen/Killjoy.
```

## 3. Найти команду

### 3.1 Основной flow

```text
Пользователь нажимает “Найти команду”
↓
Проверка: есть ли заполненная анкета
↓
Если нет — показать сообщение “Сначала заполни анкету”
↓
Если да — показать список команд
↓
Пользователь применяет фильтры
↓
Открывает команду
↓
Нажимает “Откликнуться”
↓
Пишет комментарий, необязательно
↓
Заявка отправляется капитану
↓
Капитан принимает или отклоняет
```

### 3.2 Карточка команды

Короткая карточка:

```text
👥 Team Phantom

🏆 Gold-Platinum
🎭 Нужны: Controller / Sentinel
🏙 Алматы / Онлайн
🎮 Ranked / Premier
👤 Игроков: 4/5
🎙 Микрофон: Обязательно

[Подробнее] [Откликнуться]
```

### 3.3 Подробная страница команды

```text
👥 Team Phantom

Описание:
Играем вечером, хотим стабильный стак для ranked и Premier.

🏆 Ранг: Gold-Platinum
🎭 Нужные роли: Controller / Sentinel
🏙 Город / формат: Алматы / Онлайн
🎮 Режимы: Ranked, Premier
👥 Состав: 4/5
🎙 Микрофон: Обязательно
🕒 Время: Вечером
🎯 Цель: Постоянная команда

[Откликнуться]
[Пожаловаться]
```

### 3.4 Создание команды

```text
Пользователь нажимает “Создать команду”
↓
Заполняет форму
↓
Backend создает Team
↓
Backend добавляет автора как TeamMember с is_captain=true
↓
Команда появляется в списке
```

Поля создания команды:

```text
Название
Описание
Ранг / диапазон
Нужные роли
Город
Формат: Онлайн / город / неважно
Режимы
Игроков сейчас
Сколько ищем
Микрофон
Время игры
Цель
Видимость
```

## 4. Найти игрока

### 4.1 Основной flow

```text
Пользователь нажимает “Найти игрока”
↓
Проверка: есть ли заполненная анкета
↓
Если нет — попросить заполнить анкету
↓
Если да — показать список игроков
↓
Пользователь применяет фильтры
↓
Открывает профиль игрока
↓
Нажимает “Пригласить”
↓
Пишет комментарий, необязательно
↓
Второй игрок получает входящее приглашение
↓
Принимает или отклоняет
```

### 4.2 Карточка игрока

```text
👤 ShadowPeek

🏆 Platinum
🎭 Controller / Sentinel
🏙 Алматы
🎙 Микрофон: Да
🎮 Ranked
🎯 Ищет: Дуо / Команду

[Подробнее] [Пригласить]
```

### 4.3 Подробный профиль игрока

```text
👤 ShadowPeek

🎮 Riot ID: Shadow#777
🏆 Ранг: Platinum
🎭 Роль: Controller / Sentinel
🏙 Город: Алматы
🎂 Возраст: 18-21
🎙 Микрофон: Да
🕹 Любимый режим: Ranked
🎯 Цель: Найти дуо / Найти команду

💬 О себе:
Играю вечером, не токсик.

[Пригласить сыграть]
[Пожаловаться]
```

### 4.4 Входящие/исходящие приглашения

Статусы:

```text
pending
accepted
rejected
cancelled
expired
```

Приглашение истекает через 24 часа.

## 5. Настройки

```text

```

Будущий flow:

```text
↓
Приватность
Уведомления
Город
Язык
Черный список
Удаление аккаунта/анкеты
```
